== Description ==

Adds Bootstrap styles to the WordPress TinyMCE editor's "Format" dropdown to provide easy access to style your content.

== Installation ==

1. Upload the `bootstrap-tinymce-styles` folder to the `/wp-content/plugins/` directory
1. Activate the Bootstrap TinyMCE Styles plugin through the 'Plugins' menu in WordPress
